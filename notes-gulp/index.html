gulp

流程构建工具

安装

安装gulp之前要先安装Node.js

  1. 全局安装

    npm install --global gulp

    cnpm install --global gulp(国内镜像)

  2. 局部安装

    npm install --save-dev gulp

    cnpm install --save-dev gulp(国内镜像)

  3. 必需文件gulpfile.js(配置文件)和package.js文件(依赖的插件)

  4. cnpm install(安装依赖)

  5. 重要的两个文件：

    package.js

    gulp.js
