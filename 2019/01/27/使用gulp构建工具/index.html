使用gulp构建工具

两种模式

开发模式

创建结构目录–>编译、监控、清理–>本地服务器

  • 创建目录树 gulp create
  • 清理 清理源文件 gulp clean
  • 编译 //sass文件 gulp generate
  • 开启本地服务器 gulp server
  • 监控

上线模式

压缩、打包–>上传

目录结构

|– src 源文件
— html文件
||– css
||– js
||– images

|– dist 打包文件
— html文件
||– css
||– js
||– images

|– support 第三方支持

功能

创建目录树

创建结构目录
gulp create

清理

清理源文件和清理输出的文件

gulp clean-source
gulp clean-dist

编译

编译sass文件
gulp generate

本地服务器

gulp server

监控
