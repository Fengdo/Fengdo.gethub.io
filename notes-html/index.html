基本标签

标题

<h1>-<h6>

  • 权重：h1>h2>h3>h4>h5>h6
  • h1页面的主标题，一个页面只能出现一次
  • h2副标题，可以出现多次
  • h3可作为列表或者文章的小标题也可以出现多次
  • 标题是块级元素，浏览器设有默认的样式;比如：margin,padding,font-weight……等

段落

<p>
段落是块级元素，如果不设置width限制宽度，那么默认为宽度为100%占满整个空间

换行

<br>
换行是为了保持原有的样式而存在的，一般用于段落或者想保持原有格式的文本。

水平线

<hr>

注释

<!-- 注释Start -->
<!-- 注释End -->
注释一般是成对出现的，但是有时候我们只写开始<!-- 开始 -->;

图片

<img src="" alt="" title="">

  • src：图片的链接
  • alt : 图片无法加载时的提示
  • title: 鼠标移入时的提示

表单

可选属性:

  • action : 提交地址
  • method : 提交方式; //get , post
  • id :区分表单
  • name : 提交给后台的数据的名字
  • disabled：禁用表单

文本域

<input type="text">
可选属性:

  • name : 提交给后台的数据名字
  • value : 提交给后台的数据
  • placehoder : 提示信息
