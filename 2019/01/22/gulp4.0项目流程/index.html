gulp 4.0 项目流程

准备

安装gulp前要,先检查一下版本号是否是最新，防止后续出错

检查

检查 nodejs、npm、gulp 和 gulp-cli 的版本是否是最新,如果不是就升级

  • node -v或者 node --version
  • npm -v 或者 node --version
  • gulp -v 或者 node --version
  • gulp-cli 如果没有gulp-cli或者不是最新版本可以安装或者更新 npm install --global gulp-cli

升级

  • 升级nodejs要到官网下载
  • 升级npm npm install -g npm
  • 升级gulp 可以先删除gulp:npm rm --global gulp,然后再安装 npm install -g gulp

gulp工作流程

  • 创建项目根目录: npx mkdirp myProject
  • 进入根目录: cd myProject,然后创建 package.json 配置文件 npm init
  • 安装项目依赖插件: npm install --save-dev gulp
  • 查看gulp版本信息: gulp -v ,这个时候能看到 CLI version 2.0.1 和 Local version 4.0.0
  • 创建gulpfile.js文件 touch gulpfile.js;

注意: gulpfile.js文件是忽略大小写的

命令行运行:gulp test;

文件说明

  • package.json npm项目插件依赖的配置文件
  • package-lock.json 本地的依赖文件(这个不用管它)
  • gulpfile.js 自定义的流程;比如：创建目录树，编辑文件，打包上传
