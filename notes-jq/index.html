jq笔记

Jquery

概念： 一个封装的操作dom的js的方法库。jq把所有的东西都封装在了==方法==中。

jquery是按照链式执行的。

方便快捷

口号：write less,do more

作用： 解决js代码量大、找对象麻烦、多重循环嵌套和兼容上的问题;

大纲

代码规范

了解三个环境

  1. 开发环境 —–>选择开发版
  2. 测试环境 —–>随便
  3. 生产环境 —–>选择生产版 jquery.min.js

2.x:不兼容IE6,7,8

3.x:不兼容IE6,7,8

JQ选择器

基本选择器

  1. ID选择器； 按ID来获取 $(“#btn1”);
  2. 元素选择器；按标签名获取 $(“div”);
  3. 类名选择器；按类名获取 $(“.div1”);
  4. 群组选择器；$(“td,p,div,a”);
  5. 后代选择器；$(“#div p”);
  6. 通配选择器; $(“*”);

层次选择器

  1. 集合选择器；$(“#div a”);
  2. 子代选择器；$(“#div>a”);
  3. 相邻选择器; $(“#div+p”); //id为div的下一个p元素节点；相当于==$(“#div”).next(“p”)==;
  4. 同辈选择器；$(“#div~div”); //id为div后面的所有div同辈元素节点；==$(“#div”).nextAll(“div”)==;
  5. siblings()

过滤选择器

  1. :first $(“div:first”)选取第一个div的元素
  2. :last $(“div:last”)选取最后一个div的元素
  3. :not(selector); $(“div:not(“.class1”)”);选取class不为1的div元素
  4. :even 偶数 序号是==从0开始的==
  5. :odd 奇数 序号是==从0开始的==
  6. :eq() 等于
  7. :gt() 大于
  8. :lt() 小于
  9. :header 标题
  10. :animated 正在执行动画
  11. :focus 获取焦点

内容过滤选择器

  1. :contains(text) $(“#div:contains(‘我’)”); 选择文本有”我”的元素节点
  2. :empty 不包含子元素或文本元素的空元素;
  3. :has(selector) 含有选择器所匹配的元素的元素
  4. :parent 选取邻居有子元素或都文本元素的元素

可见性过滤选择器

  1. :hidden 选取所有不可见的元素
  2. :visible 选取所有的可见的元素

属性过滤选择器

  1. $(“div[id]”)
  2. $(“div[title=test]”)
  3. $(“div[title!=test]”)
  4. $(“div[title^=test]”) 以title为test开头的
  5. $(“div[title$=test]”) 以title为test结束的
  6. $(“div[title*=test]”) 以title含有test的
  7. $(“div[title|=test]”) 以title为test或者以”test-“开头
  8. $(“div[title~=test]”) 以title以空格分隔包涵test
  9. $(“div[id][title^=test]”) 复合属性选择器

子元素过滤选择器

  1. :nth-child(index/even/odd/equation) //索引是==从1开始==计算的
    :nth-child(0) 第一个子元素
    :nth-child(even) 偶数子元素
    :nth-child(odd) 奇数子元素
    :nth-child(3n+1) 第一个子元素
  2. :first-child 第一个子元素
  3. :last-child 最后一个子元素
  4. :only-child 唯一子元素

表单选择器

  1. :enabled 选取所有可用元素 例子：$(“#form1:enabled”)选取id为form1的所有可用元素
  2. :disabled 选取所有不可用元素
  3. :checked 选取所有被选中的元素(单选框或者多选框)
  4. :selected 选取所有选中的下拉列表

表单属性过滤

    load()方法

    load方法用来加载静态的和动态的文件。

    好处:load()方法能加载选中的内容，这个$.ajax不能模拟，除非用json或其它格式。

    格式：
    $(obj).load(url,data,callback);
    obj:目标
    url:可筛选class显示
    data:是一个对象
    callback(responseText,textStatus,XMLHttpRequst)
    textStaus:success,error,timeout,notmodified 4个参数

    $.get()和$.post()

    $.get()和$.post()都是全局函数

    $.get()

    $.get(url,[data],[callback],[type])

    url：链接
    data：对象
    callback(data,textStatus)：回调函数(data)可以判断状态比

    如:if(textStatus=”success,error,timeout,notmodified”)

    type(xml,html,script,json,text,_default):数据的格式

    第一个参数是地址，第二个参数是传递的数据，第三个参数是回调函数，第四个参数是数据的格式。

    post与get的区别

    post和get用法是完全相同的，不同之处在于：

    get的数据是追加在链接url后面，是可见的。

    get只能传递不大于2k的数据

    get数据会缓存起来，有安全上的风险

    get和post服务器的获取方式不同。get用$_GET[],post用$_POST[].这两种方式都可以用$_REQUEST[]接收

    $.getScript动态加载

    $.getScript(url,callback);

    两个参数:1.url 2.callback(data,textStatus);

    传统动态加载script

    $.getScript动态加载script

    $.getJSON(记住JSON是大写的)

    json涉及到遍历对象和数组对象

    $.getJSON(url,callback);

    两个参数:1.url 2.callback(data,textStatus);

    $.each()用来遍历对象和数组对象

    $.ajax()

    $.ajax({});

    type:get/post提交方式

    url:url链接

    data:{name:”feng”,age:100};传递的数据

    dataType:返回的数据类型。getScript数据类型为script.其它的有:xml,html,text,json等

    beforeSend:function(XMLHttpRequest,textStatus){…} 发送前的效果，比如可以增加loading:正在加载中…动画

    complete:function(XMLHttpRequest,textStatus) 加载完成后，删除动画。无论结果如何都会执行

    success:function(data,textStatus) 成功

    error:function(XMLHttpRequest,textStatus,errorThrown)

    timeout:500设置超时时间

    global:true 默认为true解发全局ajax
