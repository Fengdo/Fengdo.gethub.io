关于hexo

快速、简洁且高效的博客框架
支持markdown语法

安装

  1. 准备工作

    安装前要先安装以下两个

    1. node.js
    2. Git

  2. 安装

初始化

  1. hexo init 根目录
  2. cd 根目录
  3. npm install

初始化完成后会生成以下目录与文件结构

  1. _config.yml:网站配置文件
  2. package.json:程序应用信息
  3. scaffolds:模板文件夹 //生成的文章md文件所包函的内容
  4. source:资源存放文件夹
  5. themes:主题文件夹

网站配置

  1. title 网站标题
  2. subtitle 副标题
  3. description 网站描述
  4. author 作者
  5. language 语言 //zh-cn
  6. timezone 时区
  7. url 域名
  8. root 根目录 /blog/
  9. permalink 文章永久链接格式 //:year/:month/:day/:title/
  10. permalink_defaults 永久链接默认值
