Nodejs基本命令

查看版本号

node -v或者 node --version

运行js文件

node hello.js

创建目录

npx mkdirp mydir

npm升级

npm -global(-g) install npm

npx是什么？

npx是一个npm的执行器,npx使用了CLI工具

node的两个函数 require和exports

require 入口
exports 出口

var let const

var 全局变量
let 局部变量
const 常量

安装与删除模块

install

全局安装: npm install -g <模块名>
局部安装: npm install --save-dev <模块名>

uninstall

删除全局模块

npm uninstall -g <模块名> 删除全局模块;

删除本地模块

npm uninstall <模块名> 删除模块但不删除配置文件的相关信息;
npm uninstall --save <模块名> 删除模块并删除 package.json 下 dependencies 下对应的信息;
npm uninstall -save-dev <模块名> 删除模块并删除 package.json 下 devDependencies 下对应的信息;
