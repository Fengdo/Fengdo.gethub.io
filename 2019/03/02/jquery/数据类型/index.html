数据类型

六种数据类型

  • 数字：js中的浮点数，整数都归为数字类型
  • 字符串
  • 布尔值
  • 对象:Object,Array,function,Date,RegExp
  • null
  • undefined

注意

  1. NaN也是number数据类型，只不过它是一个特殊的值表示非数字.
  2. undefined的出现有三种情况:未申明，未赋值，不存在
  3. 未获取到dom对象时会出现null
  4. typeof null会输出object，这是因为历史的问题。null最初被定义为空值的对象类型。
