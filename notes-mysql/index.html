Mysql基本命令

函数 说明
select user(); 显示当前正在使用数据库的用户名
select now(); 显示当前时间
select version(); 显示当前mysql的版本号
select * from table_name\G; 显示表的结构(在后面加G可以用竖状显示)
函数 说明
mysql -u root -p 连接数据库服务器
exit 退出据库服务器
create database database_name; 创建数据库
mysqladmin -u root -p create database_name 创建数据库(结尾不带引号)
use database_name; 选中数据库
show databases; 查看数据库列表
drop database database_name; 删除数据库
mysqladmin -u root -p drop database_name 删除数据库
show tables; 查看表的列表;(先选中数据库)
show columns from table_name; 查看表结构
desc table_name; 查看表结构

Tip:数据库不能修改名字(改)
数据库的名称带有符号时，可以使用反引号引起来，否则选不中

insert插入数据的三种方式

方法 说明
insert table_name values(“value1”,”value2”…); 如果没有写明字段就必需一一赋值;ID自增用null;
insert table_name(“a”,”b”,…) values(“value1”,”value2”,…); 字段和值对应
insert table_name set(a=”value1”,b=”value2”,…);
insert table_name() select …;

modify,change的区别

alter table table_name modify id tinyint unsigned auto_increment;

modify主要用于修改字段的属性。比如:修改数据类型，增加自增或删除自增。

alter table tabl_name change id cd varchar(30) unsigned auto_increment;

change既修改了字段的名称又修改了数据的类型。

删除表所有的数据

delete from table_name;

删除后id没有清零，会继续增长。

truncate table table_name;

删除后id也清零。
