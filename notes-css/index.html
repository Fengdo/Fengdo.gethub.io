font字体

font是以下的简写:

font[font-style,font-variant,font-weight,font-size/line-hieght,font-family]

font-style 字体风格

font-style:[normal,italic,oblique]

italic,oblique 斜体

font-variant 字体变体

font-variant [normal,small-caps,inherit]

small-caps:浏览器会显示小型大写字母的字体

inherit: 继承父级字体样式

letter-spacing 单词间距

letter-spacing:2px;

css3笔记

边框

  • border-radius 边框倒圆角

  • 兼容：ie9+

    • border-top-left-radius 左上角
    • border-top-right-radius 右上角
    • border-bottom-left-radius 左下角
    • border-bottom-right-radius 右下角

  • border-image 边框图形

  • 兼容：ie不支持

    border-image涉及到位置

阴影

  • box-shadow

  • 兼容：ie9+

box-shadow[x轴，y轴，阴影模糊值，阴影扩散半径(后在的都是可选)，阴影的颜色，inset(设置内阴影)]

x轴:阴影正数在右边，负数在左边

y轴:阴影正数在下边，负数在上边

阴影模糊值：值越大越模糊

阴影的颜色:自定义，默认为黑色

inset:内部，默认为外部

box-shadow：5px 5px 5px 5px black;

文字

  • text-shadow 文字阴影
  • text-shadow: offset-x | offset-y | blur-radius | color ;
  • offset-x: x轴偏移
  • offset-y: y轴偏移
  • blur-radius:模糊半径
  • color:阴影颜色

Tip:轻微调整文字阴影会让文字更好看

  • 例子:
    • 标准
    • text-shadow: 1px 1px 2px pink;
    • 不模糊
    • text-shadow: 5px 10px;
  • text-overflow 处理文字溢出
  • text-overflow:[ clip | ellipsis | \ ]
  • clip：截断
  • ellipsis：截断，用…代替截断的文字
  • “???”：字符串代替截断的文字只有火狐支持
  • word-break 文字换行的方式
  • word-break: normal|break-all|keep-all;
  • keep-all:保持原样
  • break-all：自动换行

自定义字体

@font-face 自定义字体

  • @font-face通过和font-family,src一起使用
    • font-family:字体的名字
    • src:找不到该字体时去指定链接下载

2D/3D转换

  • transform 转换
  • 2d
  • transform:rotate(15deg); 固定点旋转
  • transform:translate(10px,20px); 移动
  • transform:scale(2,3); 缩放
  • transform:skew(30deg,20deg); 倾斜
  • transform:matrix(); 有六个参数，包含旋转，缩放，移动（平移）和倾斜功能。
  • 3d
  • rotate3d(x,y,z,angle)
  • translate3d(x,y,z)
  • scale3d(x,y,z)

过渡

  • transition 过渡
  • 作用于伪元素before,active
  • transition:[property name | duration | timing function | delay]
  • transition:width 2s; 单项改变
  • transition:width 2s,height 2s; 多项改变
